# Tic Tac Toe

## Features

1. Built-in moves calculation

2. Winner detection

3. Reset Functionality

## Installation

1. `git clone https://github.com/madzadev/tic-tac-toe.git`

2. `cd tic-tac-toe`

3. `npm install`

4. `npm run dev`
